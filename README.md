
# Dev-Board

Welcome to **Dev-Board**, a dynamic platform designed to help developers track their projects, tasks, and ideas. Whether you're managing personal projects, collaborating with others, or simply brainstorming new ideas, Dev-Board provides the tools you need to stay organized and focused.

## Live Demo
You can explore the live version of the Dev-Board here:  
[Dev-Board Live Demo](https://riyad899.github.io/Dev-Board/)

## Features
- **Project Tracking**: Keep track of your ongoing and completed projects.
- **Task Management**: Create and manage tasks to stay organized.
- **Collaboration**: Collaborate with other developers and track progress together.
- **Idea Board**: Store and share your ideas in a visually engaging way.

## Technologies Used
- **HTML**: Structure and content of the platform.
- **CSS**: Styling and layout.
- **JavaScript**: Interactivity and dynamic elements.
- **GitHub Pages**: Hosting the live version of the project.
