What are the different ways to select an element in the DOM?

To select elements in the DOM, you can use methods like getElementById() for IDs, getElementsByClassName() for classes, and getElementsByTagName() for tags. querySelector() and querySelectorAll() allow selection using CSS selectors. For child elements, use children, and for navigating, parentNode, nextElementSibling, and previousElementSibling are useful.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent: innerHTML returns the HTML content of an element, including any HTML tags, and allows you to modify the content as HTML. innerText returns the visible text content of an element, excluding hidden text or tags, and respects CSS styles like display: none. textContent returns the text content of an element, including hidden text, and does not respect CSS styles.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM: Event delegation is a technique where a single event listener is attached to a parent element, and it listens for events on child elements. Instead of adding individual event listeners to each child, the parent captures events that bubble up and handles them, improving performance and simplifying code.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM: Event bubbling refers to the process where an event triggered on an element propagates (or "bubbles") up the DOM tree to its ancestors. For example, when you click on a button inside a div, the click event first triggers on the button and then bubbles up to the div, the body, and eventually the document.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating, adding, and removing elements using JavaScript: You can create a new element using document.createElement(), add it to the DOM using appendChild(), insertBefore(), or append(), and remove an element using removeChild() or remove(). These methods allow you to dynamically manipulate the DOM by adding, inserting, or removing elements as needed.
